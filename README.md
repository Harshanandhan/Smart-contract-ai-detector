- Carbon (https://carbon.now.sh) - Beautiful code screenshots
- Ray.so (https://ray.so) - Modern code presentation
- Snappify - Code snippets with annotations

**For Mockups:**
- Figma - UI mockups
- Canva - Quick designs
- Photopea - Free Photoshop alternative

### Screenshot Guidelines:

- **Resolution:** 1200x600 or 1920x1080 PNG
- **File Size:** < 500KB (use compression)
- **Naming:** Descriptive (e.g., `ml-pipeline-diagram.png`)
- **Background:** Dark or light (match GitHub theme)
- **Text:** Readable at all sizes

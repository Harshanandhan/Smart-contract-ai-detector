### Example 1: Reentrancy Attack

**Vulnerable Code:**
```solidity
function withdraw(uint256 amount) public {
    require(balances[msg.sender] >= amount);
    (bool success, ) = msg.sender.call{value: amount}("");  // âŒ External call
    require(success);
    balances[msg.sender] -= amount;  // âŒ State update AFTER call
}
```

**Detection Output:**
```
[1] Reentrancy
    Line 25: (bool success, ) = msg.sender.call{value: amount}("");
    Severity: CRITICAL
    Description: Potential reentrancy - external call before state update
    CWE: CWE-841
```

**Fix:**
```solidity
function withdraw(uint256 amount) public nonReentrant {
    require(balances[msg.sender] >= amount);
    balances[msg.sender] -= amount;  // âœ… State update FIRST
    (bool success, ) = msg.sender.call{value: amount}("");
    require(success);
}
```
